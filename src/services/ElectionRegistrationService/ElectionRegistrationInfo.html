SCCCC Election Registration Service

About This Service

This service provides a secure API for the Election Administrator to verify if Election Officers are active club members, without providing access to the main membership management spreadsheet.

API Endpoint

The service exposes a single endpoint to check membership status:

Function: isMember(email)

Parameters:

  • email (string) - The email address to check

Returns: 

{
  "isMember": true/false
}

Usage

This endpoint can be called from your Election Registration spreadsheet or external application to verify membership without exposing sensitive member data.
